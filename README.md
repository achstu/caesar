# caesar

The goal of this project is to implement the caesar cipher, morse code cipher and others using java.


### Building
For now supported Java 7+.
```
javac Main.java
```

### Usage
Just run
```
java Main CIPHER MODE FILE
```

Available ciphers are:
- `CAESAR`
- `MORSE`

Available modes are:
- `DECODE`
- `ENCODE`
- `PASS` (do nothing)
